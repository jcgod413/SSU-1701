### <a href="#-목차-">2.4.4</a> Bool Algebra Properties

<div style="overflow:hidden;border:solid gray;">

<div style="float:left;margin:2em">

- Commutative
  - a + b = b + a
  - a * b = b * a
- Distributive
  - a * (b + c) = a * b + a * c
  - a + (b * c) = (a + b) * (a + c)
- Associative
  - (a + b) + c = a + (b + c)
  - (a * b) * c = a * (b * c)
- Identity
  - 0 + a = a + 0 = a
  - 1 * a = a * 1 = a
</div>


<div style="float:left; margin:2em">

- Complement
  - a + a’ = 1
  - a * a’ = 0
- Null elements
  - a + 1 = 1 – a * 0 = 0
- Idempotent Law
  - a + a = a – a * a = a
- Involution Law
  - (a’)’ = a
- DeMorgan’s Law
  - (a + b)’ = a’b’
  - (ab)’ = a’ + b
</div>
